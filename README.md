## dev

`.env` を作成する

```
MICROCMS_SERVICE_DOMAIN=<Service Domain>
MICROCMS_API_KEY=<API Key>

NEWT_SPACE_UID=<Space UID>
NEWT_CDN_API_TOKEN=<CDN API Token>
```

実行！

```sh
pnpm dev

# or

pnpm build && pnpm preview
```
